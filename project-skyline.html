Cross Country Team Website

A custom, athlete-focused platform designed and built for Skyline High School Cross Country Team

Overview

This client project focused on building a user-friendly website for a local, Ann Arbor high school cross country team. The site was inspired by my experience as a student-athlete and the type of platform I always wished existed. It was created with the intentions to help runners, coaches, and fans easily access rosters, profiles, and team updates.

Key Features

  • Home Page: Includes season overview, team updates, and upcoming events
  • Roster Pages: Auto-generated pages using Python scripts that read CSV team data
  • Player Profiles: Includes profile picture, race results, seasonal stats, and comments
  • Responsive Design: Stacked mobile nav, inline desktop nav, hover animations, and skip links
  • Color Themes: Light mode themed using the school’s colors for a personalized feel
