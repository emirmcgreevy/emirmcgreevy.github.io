Detroit Tigers Statistics Website

Project Description

Built a dynamic baseball statistics website that visualizes data interactively using JSON, JavaScript, and Chart.js. Gathered MLB data in CSV format, converted it to JSON with Python, and implemented dropdown menus to let users choose different statistics. Charts automatically update based on user selections. Designed custom “memory flip cards” using CSS transform and rotateY for smooth 3D hover effects, with styled front/back containers and subtle shadow enhancements.

Key Features

  • Interactive dropdown menus for selecting different MLB statistics
  • Real-time chart updates for dynamic data visualization
  • Custom memory-style flip cards with smooth 3D hover animations
  • Responsive layout designed for desktop and mobile screens
